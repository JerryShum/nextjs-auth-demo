# Next.js Authentication Example with NextAuth.js and Middleware

This is a demo application built with **Next.js** that implements user authentication and route protection using **NextAuth.js** and **middleware**. The app features:

-  A login page using credentials-based authentication.
-  A protected dashboard that requires user login.
-  Middleware to handle route protection seamlessly.

## 🚀 Features

-  **NextAuth.js Integration**: Credential-based authentication.
-  **Protected Routes**: Middleware to restrict access.
-  **Session Management**: Access session data across the app.
-  **Responsive UI**: Built with minimal styling, ready for further customization.
